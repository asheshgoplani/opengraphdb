<objective>
Implement append-only temporal versioning for nodes with background compaction of superseded versions.

Purpose: Enable users to record explicit time-stamped snapshots of a node's properties, query them via AT TIME, and compact old versions that are no longer reachable by any time-travel query. This is the TEMP-01 requirement: temporal graphs support append-only versioning with background compaction.

Output: TemporalNodeVersion data model, add/query/compact APIs on Database, persistence via PersistedMetaStore, background compaction integration, and a test proving 1000 versioned updates compact correctly without changing AT TIME results.

**4. Implement `add_node_temporal_version()`** on Database:

```rust
pub fn add_node_temporal_version(
    &mut self,
    node_id: u64,
    valid_from: i64,
    properties: PropertyMap,
) -> Result<(), DbError> {
    let idx = node_id as usize;
    if idx >= self.node_temporal_versions.len() {
        return Err(DbError::NodeNotFound(node_id));
    }
    // Close the previous open version (set its valid_to = valid_from)
    if let Some(last) = self.node_temporal_versions[idx].last_mut() {
        if last.valid_to.is_none() {
            last.valid_to = Some(valid_from);
        }
    }
    // Append new version
    self.node_temporal_versions[idx].push(TemporalNodeVersion {
        valid_from,
        valid_to: None,
        properties,
    });
    Ok(())
}
```

**6. Implement `node_properties_at_time()`** for querying:

```rust
pub fn node_properties_at_time(&self, node_id: u64, timestamp_millis: i64) -> Option<&PropertyMap> {
    let idx = node_id as usize;
    let chain = self.node_temporal_versions.get(idx)?;
    // Find the version where valid_from <= timestamp < valid_to (or valid_to is None)
    chain.iter().find(|v| {
        v.valid_from <= timestamp_millis
            && v.valid_to.map_or(true, |to| timestamp_millis < to)
    }).map(|v| &v.properties)
}
```

**7. Implement `compact_temporal_versions()`**:

```rust
pub fn compact_temporal_versions(&mut self, compaction_floor_millis: i64) -> Result<usize, DbError> {
    let mut total_removed = 0;
    for chain in &mut self.node_temporal_versions {
        let before = chain.len();
        chain.retain(|v| {
            // Keep if: open-ended (current), or valid_to is after the floor
            v.valid_to.is_none() || v.valid_to.unwrap() > compaction_floor_millis
        });
        total_removed += before - chain.len();
    }
    Ok(total_removed)
}
```
