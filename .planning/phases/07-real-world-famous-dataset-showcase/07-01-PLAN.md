  <action>
    Create 4 download shell scripts and 4 Python conversion scripts. Each download script caches raw files in `data/cache/` (create directory if needed, skip download if cache file exists). Each conversion script reads from `data/cache/` and writes to `datasets/<name>.json`.

    **Download scripts (bash, executable):**

    1. `download-movielens.sh`: Download `https://files.grouplens.org/datasets/movielens/ml-25m.zip` to `data/cache/ml-25m.zip`. Extract `ml-25m/movies.csv` and `ml-25m/ratings.csv` to `data/cache/`. Use `curl -L -o` for download, Python `zipfile` or `unzip` for extraction. Skip if `data/cache/ml-25m/movies.csv` exists.

    2. `download-airroutes.sh`: Download two CSVs from GitHub raw URLs:
       - `https://raw.githubusercontent.com/krlawrence/graph/master/sample-data/air-routes-latest-nodes.csv` to `data/cache/air-routes-latest-nodes.csv`
       - `https://raw.githubusercontent.com/krlawrence/graph/master/sample-data/air-routes-latest-edges.csv` to `data/cache/air-routes-latest-edges.csv`

    3. `download-got.sh`: Download 16 CSV files (2 per season) from GitHub raw URLs:
       - `https://raw.githubusercontent.com/mathbeveridge/gameofthrones/master/data/got-s{1..8}-nodes.csv`
       - `https://raw.githubusercontent.com/mathbeveridge/gameofthrones/master/data/got-s{1..8}-edges.csv`
       All to `data/cache/`.

    4. `download-wikidata.sh`: Call `https://api.nobelprize.org/2.1/laureates?limit=2000` and save to `data/cache/nobel-laureates.json`. Use `curl -o`.

    All download scripts must: `set -euo pipefail`, create `data/cache/` if missing, skip download if target file exists (print "Already cached"), be executable (`chmod +x`).

    **Conversion scripts (Python 3 stdlib only):**

    Use globally unique non-overlapping ID ranges to prevent collision when all datasets are loaded into the same database:
    - MovieLens: node IDs 0-99999, edge IDs 0-99999
    - Air Routes: node IDs 1_000_000+, edge IDs 1_000_000+
    - GoT: node IDs 2_000_000+, edge IDs 2_000_000+
    - Wikidata: node IDs 3_000_000+, edge IDs 3_000_000+

    1. `convert-movielens.py`: Read `data/cache/ml-25m/movies.csv` and `data/cache/ml-25m/ratings.csv`. Select top ~8000 movies by rating count. Create Movie nodes with properties: `title`, `released` (parse year from title e.g. "Toy Story (1995)"), `genres` (first genre as string), `avgRating` (float), `ratingCount` (int), `_label: "Movie"`, `_dataset: "movielens"`. Create Genre nodes (unique genres). Create `IN_GENRE` edges from Movie to Genre. Do NOT create User nodes. Write to `datasets/movielens.json`.

    2. `convert-airroutes.py`: Read `data/cache/air-routes-latest-nodes.csv` and `data/cache/air-routes-latest-edges.csv`. Create Airport nodes with `code`, `icao`, `name` (from desc column), `city`, `country`, `region`, `lat` (float), `lon` (float), `runways` (int), `elev` (int). Create Country and Continent nodes. Create `route` edges with `dist` property. Create `contains` edges. CRITICAL: lat and lon must be preserved as float properties. Skip the version metadata row (first row where ~label is "version"). Use `_dataset: "airroutes"` on all nodes.

    3. `convert-got.py`: Read all 8 seasons of node and edge CSVs from `data/cache/got-s{1..8}-nodes.csv` and `got-s{1..8}-edges.csv`. Deduplicate Character nodes using `Id` column. Create 8 Season nodes (Season 1 through Season 8). Create `APPEARS_IN` edges from characters to seasons they appear in. Create `INTERACTS` edges with `weight` (int) and `season` (int) properties. Character properties: `name` (from Label column), `characterId` (from Id column). Use `_dataset: "got"`.

    4. `convert-wikidata.py`: Read `data/cache/nobel-laureates.json`. Create Laureate nodes with `name`, `gender`, `birthYear`, `birthCountry`, `wikidataId`. Create Category nodes (6 Nobel categories). Create Country nodes (from birth place countryNow). Create Institution nodes (top 50 most common affiliations). Create `WON_PRIZE_IN` edges with `year` property. Create `BORN_IN` edges to Country. Create `AFFILIATED_WITH` edges from Laureate to Institution. Use `_dataset: "wikidata"`.

    All conversion scripts must: use Python 3 stdlib only (`csv`, `json`, `urllib.request`, `zipfile`), write formatted JSON with `indent=2`, print summary stats (node count, edge count) after writing.

    After creating all scripts, run the download scripts and then the conversion scripts to generate the 4 dataset JSON files. Verify each JSON file is valid and contains the expected `_dataset` property.

    Add `data/cache/` to `.git/info/exclude` if not already gitignored (the `data/` directory is already in `.gitignore` per Phase 6 summary).

    Also delete the old synthetic dataset files that are being replaced: `datasets/movies.json`, `datasets/social.json`, `datasets/fraud.json`. Use `trash` command (not `rm`).
