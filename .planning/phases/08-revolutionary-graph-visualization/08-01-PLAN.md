**Component implementation:**
- Extract airports (nodes with label 'Airport') from graphData
- Compute connection counts per airport (number of routes)
- Build a `ScatterplotLayer` for airports:
  - `getPosition`: `[node.properties.lon, node.properties.lat]` (cast to number)
  - `getRadius`: base 5000 + connections * 2000
  - `getFillColor`: `[0, 200, 255, 220]` (cyan glow)
  - `radiusMinPixels: 3`, `radiusMaxPixels: 18`
  - `pickable: true`
  - `onClick`: call `selectNode(airport.id)` from the graph store
- Build an `ArcLayer` for routes with subtle directional animation:
  - `getSourcePosition` / `getTargetPosition`: look up airport lat/lon from a Map keyed by node ID
  - `getSourceColor`: `[0, 128, 255, 80]`
  - `getTargetColor`: `[0, 200, 255, 80]`
  - `getWidth: 1`
  - `getHeight: 0.5` (arc curvature)
  - `greatCircle: true` for Earth-curvature arcs
  - **Arc animation (LOCKED DECISION):** Add a pulsing opacity animation using a `useEffect` + `requestAnimationFrame` loop that cycles a `pulsePhase` state variable (0 to 1). Use this to modulate arc opacity via a `getTilt` or by cycling `getSourceColor`/`getTargetColor` alpha channel. Implementation:
    ```typescript
    const [pulsePhase, setPulsePhase] = useState(0)
    useEffect(() => {
      let raf: number
      const animate = () => {
        setPulsePhase((prev) => (prev + 0.003) % 1)
        raf = requestAnimationFrame(animate)
      }
      raf = requestAnimationFrame(animate)
      return () => cancelAnimationFrame(raf)
    }, [])
    ```
    Then in ArcLayer props, use `updateTriggers: { getSourceColor: pulsePhase, getTargetColor: pulsePhase }` and modulate the alpha channel based on `pulsePhase` to create a gentle directional pulse:
    ```typescript
    getSourceColor: [0, 128, 255, Math.floor(40 + 40 * Math.sin(pulsePhase * Math.PI * 2))]
    getTargetColor: [0, 200, 255, Math.floor(60 + 40 * Math.sin(pulsePhase * Math.PI * 2))]
    ```
    This creates a subtle breathing/pulse effect on the route arcs per the locked decision "Route arcs should have subtle animation (directional flow or pulse)."
- Wrap in a `<Map>` component:
  - `initialViewState`: `{ longitude: 0, latitude: 30, zoom: 1.5, pitch: 0 }`
  - `mapStyle={DARK_MAP_STYLE}`
  - `style={{ width: '100%', height: '100%' }}`
- Include `<DeckGLOverlay layers={layers} />` inside the Map
- Handle hover: show a tooltip div positioned absolutely with airport code + city

Use `useMemo` for layers to prevent re-creation on every render. Use `useCallback` for event handlers.
