Create `frontend/src/stores/ai-chat.ts` with an ephemeral Zustand store (NO persist middleware). This store manages session-only chat state.

```typescript
import { create } from 'zustand'

interface AIChatMessage {
  id: string
  role: 'user' | 'assistant' | 'system'
  content: string
  isStreaming: boolean
  cypherBlocks: string[]      // populated after streaming completes via extractCypherBlocks
  queryResult?: string        // summary of query execution result (fed back to AI)
  queryError?: string         // error message if query execution failed
}

interface DownloadProgress {
  message: string
  percent: number
}

interface AIChatState {
  messages: AIChatMessage[]
  isOpen: boolean
  isLoading: boolean
  downloadProgress: DownloadProgress | null

  setIsOpen: (open: boolean) => void
  addUserMessage: (content: string) => string             // returns message id
  startAssistantMessage: () => string                     // returns message id
  appendToMessage: (id: string, token: string) => void
  finalizeMessage: (id: string, cypherBlocks: string[]) => void
  setQueryResult: (id: string, result: string) => void
  setQueryError: (id: string, error: string) => void
  setIsLoading: (loading: boolean) => void
  setDownloadProgress: (progress: DownloadProgress | null) => void
  clearMessages: () => void
}
```

Implementation details:
- `addUserMessage`: generates a unique id (use `crypto.randomUUID()` or `Date.now().toString(36)`), pushes a new message with `role: 'user'`, `isStreaming: false`, returns the id.
- `startAssistantMessage`: generates an id, pushes `role: 'assistant'`, `content: ''`, `isStreaming: true`, returns the id.
- `appendToMessage`: finds message by id, appends token to content. Use immer-style or spread to avoid mutation.
- `finalizeMessage`: sets `isStreaming: false` and populates `cypherBlocks` array.
- `setQueryResult`: sets `queryResult` on the specified message.
- `setQueryError`: sets `queryError` on the specified message.
- `clearMessages`: resets messages to empty array.
- `setIsOpen`: toggles the panel open/close.

Export as `useAIChatStore`.

```tsx
interface AIChatMessageProps {
```

Implementation:

- **Assistant messages**: Render left-aligned with no bubble background (cleaner look). Use `Streamdown` from the `streamdown` package for markdown rendering:

  This handles streaming partial content (unterminated code blocks) gracefully.
- **Action buttons**: After streaming completes (`isStreaming === false`) and `cypherBlocks.length > 0`, render action buttons below the message for EACH Cypher block:
  - "Run query" button: `<Button variant="outline" size="sm">` with Play icon from lucide-react. Calls `onRunQuery(cypherBlock)`.
  - "Copy" button: `<Button variant="ghost" size="sm">` with Copy icon. Uses `navigator.clipboard.writeText(cypherBlock)`. Show brief "Copied!" feedback (swap icon/text for 2 seconds).
  - Disable "Run query" while `isStreaming === true` (per Pitfall 4 in research: code block extraction before streaming completes).
- **Error feedback**: If `message.queryError` is set, show a red-tinted error message below the Cypher block: "Query failed: {error}".
- **Result summary**: If `message.queryResult` is set, show a subtle success note: "{queryResult}".
