**Create `skills/README.md`**

The README is the primary onboarding document. Write it for developers who want to add OpenGraphDB expertise to their AI coding tool.

**Section 1: Header**
```
# @opengraphdb/skills

AI coding skills for OpenGraphDB. Install expert graph database knowledge into Claude Code, Cursor, VS Code Copilot, and Codex.
```

**Section 2: What Are Skills?**
Brief explanation (3-4 lines): Skills are instruction packages that teach AI coding tools how to work with OpenGraphDB. They include Cypher query patterns, exploration strategies, schema design guidance, and data import assistance. Unlike generic LLM knowledge, these skills are tuned specifically for OpenGraphDB's dialect and features.

**Section 3: Quickstart**
```bash
# Install all skills (auto-detects your AI tool)
npx @opengraphdb/skills install

# Or specify your platform
npx @opengraphdb/skills install claude
npx @opengraphdb/skills install cursor
npx @opengraphdb/skills install copilot
npx @opengraphdb/skills install codex
```

**Section 4: Available Skills**

Table with 4 skills:

| Skill | What It Does | Example |
|-------|-------------|---------|
| `ogdb-cypher` | Generates correct Cypher queries for OpenGraphDB | "Find all people who acted in movies directed by Spielberg" |
| `graph-explore` | Guides systematic graph exploration | "Help me understand what's in this database" |
| `schema-advisor` | Designs graph schemas from domain descriptions | "Design a schema for a healthcare system" |
| `data-import` | Assists CSV/JSON/RDF import with validation | "Import this CSV of employees into the graph" |

**Section 5: Platform Configuration Details**

For each platform, show what happens after `install`:

**Claude Code:**
- Creates `.claude/skills/ogdb-cypher/SKILL.md` + `rules/*.md`
- Creates `.claude/skills/graph-explore/SKILL.md` + `rules/*.md`
- Creates `.claude/skills/schema-advisor/SKILL.md` + `rules/*.md`
- Creates `.claude/skills/data-import/SKILL.md` + `rules/*.md`
- Claude Code automatically reads SKILL.md files from `.claude/skills/`

**Cursor:**
- Creates/appends to `.cursorrules` with all skill content
- Cursor reads `.cursorrules` on project open

**VS Code Copilot:**
- Creates `.github/copilot-instructions.md` with all skill content
- Copilot reads project instructions from this file

**Codex:**
- Creates `.codex/instructions.md` with all skill content

**Section 6: Using With MCP Server**

Skills work best when combined with the @opengraphdb/mcp server. Show the recommended setup:

```bash
# 1. Install MCP server
npx @opengraphdb/mcp  # Add to your AI tool's MCP config

# 2. Install skills
npx @opengraphdb/skills install

# 3. Start OpenGraphDB
opengraphdb serve mydb.ogdb --http

# Now your AI tool can:
# - Understand graph database concepts (skills)
# - Execute queries against your database (MCP)
```

**Section 7: Running Evals**

