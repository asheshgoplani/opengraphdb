2. Create `frontend/src/data/sampleGraph.ts` (TDD green phase):
   - Import `type { GraphData, GraphNode }` from `@/types/graph`
   - Export `type PlaygroundQueryKey = 'all' | 'movies-only' | 'actors-only' | 'acted-in' | 'directed'`
   - Export `const MOVIES_SAMPLE: GraphData` with a curated subset of the Neo4j movies graph:
     - 5-6 movies: The Matrix (1999), The Matrix Reloaded (2003), The Matrix Revolutions (2003), Jerry Maguire (1996), Top Gun (1986), A Few Good Men (1992)
     - 10-12 people: Keanu Reeves, Laurence Fishburne, Carrie-Anne Moss, Lana Wachowski, Lilly Wachowski, Tom Cruise, Cuba Gooding Jr., Cameron Crowe, Tom Hanks, Meg Ryan, Tony Scott, Rob Reiner

     - Links covering all three types: ACTED_IN (with roles property), DIRECTED, WROTE
     - Use numeric IDs starting from 1

   - Create helper `function filterByRelationshipType(type: string): GraphData`:
     - Filter MOVIES_SAMPLE.links to only those matching the type
     - Collect node IDs from filtered links (handling that source/target could be objects after d3 mutation: use `typeof l.source === 'object' ? (l.source as GraphNode).id : l.source`)
     - Filter MOVIES_SAMPLE.nodes to only those whose id is in the collected set
     - Return NEW objects via `.map(n => ({ ...n }))` for nodes and `.map(l => ({ ...l }))` for links (prevents d3 mutation aliasing)

   - Export `function runPlaygroundQuery(key: PlaygroundQueryKey): GraphData`:
     - 'all': return `{ nodes: MOVIES_SAMPLE.nodes.map(n => ({ ...n })), links: MOVIES_SAMPLE.links.map(l => ({ ...l })) }`

     - 'actors-only': return `{ nodes: MOVIES_SAMPLE.nodes.filter(n => n.labels.includes('Person')).map(n => ({ ...n })), links: [] }`
     - 'acted-in': return `filterByRelationshipType('ACTED_IN')`
     - 'directed': return `filterByRelationshipType('DIRECTED')`
