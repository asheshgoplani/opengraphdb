2. Define GUIDED_QUERIES constant array inside the module (not exported):
   ```typescript
   const GUIDED_QUERIES: { key: PlaygroundQueryKey; label: string; cypher: string }[] = [
     { key: 'all',         label: 'All nodes',       cypher: 'MATCH (n) RETURN n LIMIT 50' },
     { key: 'acted-in',    label: 'Actors & movies',  cypher: 'MATCH (p:Person)-[:ACTED_IN]->(m:Movie) RETURN p, m' },
     { key: 'directed',    label: 'Directors',        cypher: 'MATCH (p:Person)-[:DIRECTED]->(m:Movie) RETURN p, m' },
     { key: 'movies-only', label: 'Movies only',      cypher: 'MATCH (m:Movie) RETURN m' },
   ]
   ```

3. Export DEFAULT function `PlaygroundPage` (MUST be default export for React.lazy in AppRouter):
   - State: `const [graphData, setGraphData] = useState<GraphData>(runPlaygroundQuery('all'))` — initialize with 'all' query (using runPlaygroundQuery, NOT direct MOVIES_SAMPLE reference, to get fresh objects)
   - State: `const [activeKey, setActiveKey] = useState<PlaygroundQueryKey>('all')`
   - Handler: `const handleQuery = (key: PlaygroundQueryKey) => { setActiveKey(key); setGraphData(runPlaygroundQuery(key)) }`

4. JSX structure:
   ```tsx
   <div className="flex flex-col h-screen bg-background text-foreground">
     {/* Navigation bar */}
     <nav className="h-12 border-b bg-card flex items-center px-3 sm:px-4 gap-3">
       <Button variant="ghost" size="sm" asChild>
         <Link to="/"><ArrowLeft className="h-4 w-4 mr-1" /> Back</Link>
       </Button>
       <span className="font-semibold text-sm">Playground</span>
       <span className="text-xs text-muted-foreground hidden sm:inline">Sample movies graph</span>
     </nav>

     {/* Query buttons bar */}
     <div className="border-b bg-card/50 px-3 sm:px-4 py-2.5 flex flex-wrap items-center gap-2">
       {GUIDED_QUERIES.map(q => (
         <Button
           key={q.key}
           variant={activeKey === q.key ? 'default' : 'outline'}
           size="sm"
           onClick={() => handleQuery(q.key)}
         >
           {q.label}
         </Button>
       ))}
       <code className="ml-auto text-xs text-muted-foreground hidden md:block font-mono">
         {GUIDED_QUERIES.find(q => q.key === activeKey)?.cypher}
       </code>
     </div>

     {/* Graph canvas - takes remaining space */}
     <div className="flex-1 overflow-hidden">
       <GraphCanvas graphData={graphData} />
     </div>
   </div>
   ```

IMPORTANT:
- Do NOT import or use `useCypherQuery` — playground never calls the real API
- Do NOT store graph state in `useGraphStore` — playground state is local via useState
- Do NOT use `MOVIES_SAMPLE` directly as initial state — use `runPlaygroundQuery('all')` to get fresh objects that won't be mutated by react-force-graph-2d
- GraphCanvas receives `graphData` prop directly — the existing component handles all rendering, drag, zoom, and click behavior
