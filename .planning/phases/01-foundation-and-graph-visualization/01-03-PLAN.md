       - Light: bg '#ffffff', text '#1a1a1a', edge '#cccccc', border '#e0e0e0', nodeText '#1a1a1a'

    2. Create frontend/src/components/graph/NodeRenderer.ts:
       - Export LABEL_COLORS: array of 12 distinct colors that work in both light and dark mode
         ['#6366f1', '#ec4899', '#f59e0b', '#10b981', '#3b82f6', '#8b5cf6', '#ef4444', '#14b8a6', '#f97316', '#06b6d4', '#84cc16', '#a855f7']
       - Export getLabelColor(label, labelIndex): assigns consistent color per label using a Map
       - Export paintNode(node, ctx, globalScale, colors, labelIndex): draws a circle with label color, subtle border, and text label below
         - Circle radius: 6
         - Font: scales with globalScale, min 2px
         - Label text: truncated to 15 chars, positioned below node
         - Use colors param for border and text colors

    3. Create frontend/src/components/graph/GraphCanvas.tsx:
       - Accept graphData: GraphData prop
       - Use ForceGraph2D from react-force-graph-2d
       - Use useGraphColors() for canvas colors
       - Use useCallback for nodeCanvasObject that calls paintNode
       - Use useMemo for labelIndex (Map<string, number>) to keep color assignments stable
       - Props configuration:
         - backgroundColor: colors.bg
         - nodeCanvasObject: paintNode wrapper
         - nodeCanvasObjectMode: () => 'replace'
         - linkColor: () => colors.edge
         - linkDirectionalArrowLength: 4
         - linkDirectionalArrowRelPos: 1
         - linkLabel: (link) => link.type (shows relationship type on hover)
         - linkCanvasObjectMode: () => 'after' (for edge labels)
         - onNodeClick: calls graphStore.selectNode(node.id)
         - onLinkClick: calls graphStore.selectEdge(link.id)
         - onBackgroundClick: calls graphStore.clearSelection()
         - enableNodeDrag: true (default)
         - enableZoomInteraction: true (default)
         - d3AlphaDecay: 0.02
         - d3VelocityDecay: 0.3
         - cooldownTicks: 100
       - Handle container resizing: use a ref on the container div, listen to ResizeObserver, pass width and height to ForceGraph2D
       - Container: div with ref, w-full h-full

    4. Create a helper function to transform QueryResponse to GraphData:
       - Map response.nodes to GraphNode: set label to first labels entry or id.toString()
       - Map response.relationships to GraphEdge: set source to startNode, target to endNode
       - Export as transformQueryResponse(response: QueryResponse): GraphData
       - Place in frontend/src/api/transform.ts or within GraphCanvas

       - Layout: AppShell wraps everything, top section has QueryInput + QueryError, main area has GraphCanvas
       - Empty state: when no query result, show centered message "Run a query to see results" with example query
       - When result exists, show GraphCanvas with transformed data
