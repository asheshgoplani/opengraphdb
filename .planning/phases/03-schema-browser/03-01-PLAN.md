    2. Create directory `frontend/src/components/schema/` if it does not exist.

    3. Create `frontend/src/components/schema/SchemaPanel.tsx`:

       Imports:
       - `Sheet`, `SheetContent`, `SheetHeader`, `SheetTitle`, `SheetTrigger`, `SheetDescription` from `@/components/ui/sheet`
       - `Accordion`, `AccordionContent`, `AccordionItem`, `AccordionTrigger` from `@/components/ui/accordion`
       - `Button` from `@/components/ui/button`
       - `Badge` from `@/components/ui/badge`
       - `Database`, `RefreshCw`, `Tag`, `ArrowRightLeft`, `Key` from `lucide-react`
       - `useSchemaQuery` from `@/api/queries`
       - `cn` from `@/lib/utils`

       Component: `export function SchemaPanel()`

       Body:
       - Call `useSchemaQuery()` to get `{ data, isFetching, isError, error, refetch }`

       Render structure:
       ```tsx
       <Sheet>
         <SheetTrigger asChild>
           <Button variant="ghost" size="icon" title="Schema Browser">
             <Database className="h-4 w-4" />
           </Button>
         </SheetTrigger>
         <SheetContent side="left" className="w-80 overflow-y-auto">
           <SheetHeader>
             <div className="flex items-center justify-between">
               <SheetTitle>Schema</SheetTitle>
               <Button
                 variant="ghost"
                 size="icon"
                 onClick={() => refetch()}
                 disabled={isFetching}
                 title="Refresh schema"
               >
                 <RefreshCw className={cn('h-4 w-4', isFetching && 'animate-spin')} />
               </Button>
             </div>
             <SheetDescription>Browse database node labels, relationship types, and property keys.</SheetDescription>
           </SheetHeader>

           {/* Error state */}
           {isError && (
             <div className="mt-4 rounded-md border border-destructive/50 bg-destructive/10 p-3 text-sm text-destructive">
               <p className="font-medium">Could not load schema</p>
               <p className="mt-1 text-muted-foreground">
                 {error instanceof Error ? error.message : 'Is the backend connected?'}
               </p>
             </div>
           )}

           {/* Schema content */}
           {!isError && (
             <Accordion type="multiple" defaultValue={['labels', 'relationships', 'properties']} className="mt-4">
               <AccordionItem value="labels">
                 <AccordionTrigger>
                   <span className="flex items-center gap-2">
                     <Tag className="h-4 w-4 text-muted-foreground" />
                     Node Labels
                     <Badge variant="secondary">{data?.labels.length ?? 0}</Badge>
                   </span>
                 </AccordionTrigger>
                 <AccordionContent>
                   <ul className="space-y-1 text-sm">
                     {data?.labels.map((label) => (
                       <li key={label} className="rounded px-2 py-1 hover:bg-muted">{label}</li>
                     ))}
                     {!data?.labels.length && (
                       <li className="italic text-muted-foreground">No labels found</li>
                     )}
                   </ul>
                 </AccordionContent>
               </AccordionItem>

               <AccordionItem value="relationships">
                 <AccordionTrigger>
                   <span className="flex items-center gap-2">
                     <ArrowRightLeft className="h-4 w-4 text-muted-foreground" />
                     Relationship Types
                     <Badge variant="secondary">{data?.relationshipTypes.length ?? 0}</Badge>
                   </span>
                 </AccordionTrigger>
                 <AccordionContent>
                   <ul className="space-y-1 text-sm">
                     {data?.relationshipTypes.map((type) => (
                       <li key={type} className="rounded px-2 py-1 hover:bg-muted">{type}</li>
                     ))}
                     {!data?.relationshipTypes.length && (
                       <li className="italic text-muted-foreground">No relationship types found</li>
                     )}
                   </ul>
                 </AccordionContent>
               </AccordionItem>

               <AccordionItem value="properties">
                 <AccordionTrigger>
                   <span className="flex items-center gap-2">
                     <Key className="h-4 w-4 text-muted-foreground" />
                     Property Keys
                     <Badge variant="secondary">{data?.propertyKeys.length ?? 0}</Badge>
                   </span>
                 </AccordionTrigger>
                 <AccordionContent>
                   <ul className="space-y-1 text-sm">
                     {data?.propertyKeys.map((key) => (
                       <li key={key} className="rounded px-2 py-1 hover:bg-muted">{key}</li>
                     ))}
                     {!data?.propertyKeys.length && (
                       <li className="italic text-muted-foreground">No property keys found</li>
                     )}
                   </ul>
                 </AccordionContent>
               </AccordionItem>
             </Accordion>
           )}
         </SheetContent>
       </Sheet>
       ```

       IMPORTANT notes:
       - Use `side="left"` on SheetContent, NOT `side="right"`. The PropertyPanel already opens from the right for node/edge inspection. Using left avoids two overlapping right-side sheets.
       - Wrap the label text and Badge inside a `<span className="flex items-center gap-2">` within AccordionTrigger. The trigger has its own flex layout with a chevron; extra elements outside a span break the chevron alignment.
       - Include `SheetDescription` to avoid the Radix accessibility warning about missing description.
       - Include error state: when `isError` is true, render a styled error box with the error message and keep the refresh button accessible.
       - The `data?.labels.length` checks handle both undefined data (loading) and empty arrays (no items in database).
  </action>
